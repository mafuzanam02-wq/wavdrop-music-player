# Wavdrop Feature Backlog

- **Export before reset**: Wavdrop has no in-app reset/clear-data feature yet. If one is added,
  it must prompt the user to export a backup first (and ideally run Back Up Now automatically)
  before any destructive action. Until then, all data-loss risk comes from OS-level
  uninstall/clear-data — covered by the backup warnings in SOFT_LAUNCH_QA_CHECKLIST.md.
